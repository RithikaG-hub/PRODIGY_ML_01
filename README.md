# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project is developed as part of my internship at Prodigy InfoTech.

The goal is to predict house prices based on:
- Square Footage (GrLivArea)
- Number of Bedrooms (BedroomAbvGr)
- Number of Bathrooms (FullBath)

---

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

---

## 📊 Machine Learning Model
We used **Linear Regression** to train the model and predict house prices.

---

## 🚀 How it Works
1. Load dataset
2. Select important features
3. Train the model using Linear Regression
4. Predict house price based on input values

---

## 💡 Sample Input
- Area: 2000 sqft  
- Bedrooms: 3  
- Bathrooms: 2  

👉 The model predicts the house price based on these values.

## 📊 Output

R2 Score: 0.75  
Mean Squared Error: 1200000000  
Predicted Price: 180000

---

## 📂 Dataset
Dataset taken from Kaggle House Prices Competition.
